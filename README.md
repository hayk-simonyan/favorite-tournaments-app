## How to run this application locally

1. git clone https://github.com/hayk77/favorite-tournaments-app
2. `cd /tournaments-search-backend`
3. run `npm i`
4. run `npm start`
5. `cd /tournaments-search-frontend`
6. run `npm i`
7. run `npm start`
